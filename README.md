# Quantitative Finance Portfolio

Welcome to my quantitative finance project portfolio.
This repository contains financial models and data analysis projects developed using Python as part of my learning journey in the Master of Science in Financial Engineering.

## Projects

### Yield Curve Modeling

Implementation of the Nelson-Siegel model to analyze and visualize interest rate term structures using historical treasury yield data.

### Portfolio Optimization

Construction of an optimal investment portfolio using Modern Portfolio Theory to minimize portfolio risk while maintaining diversification.

### PCA Interest Rate Analysis

Application of Principal Component Analysis to identify key factors that drive movements in the yield curve.

### Risk Management (Value-at-Risk)

Development of a financial risk model to estimate potential portfolio losses using Value-at-Risk methodology.

### Algorithmic Trading Strategy

Design and backtesting of a simple quantitative trading strategy based on moving average indicators.
